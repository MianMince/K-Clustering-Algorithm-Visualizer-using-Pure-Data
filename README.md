# K-Means Clustering

K-means clustering is an unsupervised machine learning algorithm used to partition a dataset into distinct clusters based on feature similarity. The algorithm aims to minimize variance within each cluster and maximize the distance between different clusters.

## Program Features
- Interactive visualization of K-means clustering.
- Adjustable parameters for K value and iterations.
- Support for different datasets.
- Visual feedback on clustering performance with real-time updates.
